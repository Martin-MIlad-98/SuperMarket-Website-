# 🛒 Supermarket Management Website

## 📌 Project Overview  
Welcome to the **Supermarket Management System** — a web-based application designed to digitize supermarket operations for both **customers** and **administrators**.

The system replaces manual processes with a faster, more organized, and user-friendly platform. Customers can browse products, manage carts, and place orders, while admins can control inventory, products, and users.

---

## 🎯 Features  

### 👤 Customer Features  
- ✅ User registration & secure login  
- ✅ Browse products by categories (groceries, drinks, snacks, etc.)  
- ✅ Search & filter products  
- ✅ Add items to cart  
- ✅ View & edit cart  
- ✅ Checkout & payment simulation  
- ✅ View order history  

### 🛠️ Admin Features  
- ✅ Admin dashboard login  
- ✅ Product management (add / update / delete)  
- ✅ Inventory management  
- ✅ Order management (update status)  
- ✅ User management

## ⚙️ How the System Works  

### 👤 Customer Flow  
1. Register / Login  
2. Browse & search products  
3. Add items to cart  
4. Checkout  
5. Order is created after payment  

### 🛠️ Admin Flow  
1. Login to dashboard  
2. Manage products & inventory  
3. Manage users  
4. Track and update orders
